# Vimコマンド

## カーソルの移動・テキスト編集

|コマンド|説明|
|---|---|
|h|左に移動|
|j|下に移動|
|k|上に移動|
|l|右に移動|
|:q!|変更を破棄|
|:wq|変更を保存|
|i|カーソル位置にテキストを追加|
|A|行末にテキストを追加|

## ビジュアルモード

|コマンド|説明|
|---|---|
|v|ビジュアル選択モード|
|Ctrl + v|ビジュアル選択モード(矩形)|
|Esc|ビジュアル選択モード解除|
|gv|直前まで選択していた状態に復帰する|

## その他

|コマンド|説明|
|---|---|
|Ctrl + [|インサートモード時のEscキーと同じ|
